# 1 Введение

Данный пример показывает базовые возможности конвертера Markdown → DOCX по требованиям БГУИР. Текст абзаца выравнивается по ширине, применяется шрифт Times New Roman 14 pt и абзацный отступ.

## 1.1 Цели работы

- Показать структуру разделов.
- Продемонстрировать списки и формулы.

## 1.2 Пример формулы

Текст с встроенной формулой $E=mc^2$ внутри строки.

$$
S = \pi r^2
$$
где S - площадь
r - радиус
## 1.3 Пример кода

```python
def area(radius: float) -> float:
    return 3.14159 * radius ** 2
```
